### Installation

Project requires [Node.js](https://nodejs.org/) to run.

You need Gulp installed globally:

```sh
$ npm i -g gulp
```

```sh
$ git clone [https://github.com/Katy-Kate/Royal-event-landing-.git]
$ npm i 
$ gulp (if you want to change something)
```
